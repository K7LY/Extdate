# Unity 2D Agricola風ワーカープレイスメントゲーム

このプロジェクトは、Unity エンジンを使用して作成された、名作ボードゲーム「Agricola」をベースにした2Dワーカープレイスメントゲームです。

## 🎮 ゲームの概要

プレイヤーは農家となり、家族（ワーカー）を様々なアクションスペースに配置して、農場を発展させ、動物を飼い、家族を養いながら最も繁栄した農場を目指すゲームです。

### 主な要素
- **家族（ワーカー）**: 最初は夫婦2人、成長により最大5人まで増加
- **アクションスペース**: 家族を配置してアクションを実行する場所
- **住居**: 木の家から始まり、土の家、石の家に改築可能
- **農場**: 畑を耕し、穀物や野菜を栽培
- **牧畜**: 柵を建て、羊・猪・牛を飼育
- **リソース**: 木材、土、葦、石、穀物、野菜、食料
- **収穫フェーズ**: 定期的に食料で家族を養う必要

## 🎯 ゲームの流れ（Agricola風）

1. **ゲーム開始**: 各プレイヤーは夫婦2人と木の家2部屋から開始
2. **ラウンドフェーズ**: プレイヤーが順番に家族をアクションスペースに配置
3. **アクション実行**: 配置された家族に対応するアクションを実行
4. **ラウンド終了**: 全プレイヤーが家族を配置し終えたらラウンド終了
5. **収穫フェーズ**: 4,7,9,11,13,14ラウンド後に発生
   - 作物の収穫
   - 家族への食料供給（1人につき2食料）
   - 動物の繁殖
6. **ゲーム終了**: 14ラウンド終了後、最終スコア計算

## 🛠️ セットアップ方法

### 必要な環境
- Unity 2022.3 LTS 以降
- TextMeshPro パッケージ

### インストール手順
1. Unity でプロジェクトを作成
2. 本プロジェクトのファイルをコピー
3. `GameSetup.cs` をシーンに配置してゲームを初期化

### 基本的なシーンセットアップ
1. 空のGameObjectを作成し、`GameManager` コンポーネントを追加
2. 空のGameObjectを作成し、`GameSetup` コンポーネントを追加
3. UICanvas を作成し、`GameUI` コンポーネントを追加
4. カメラを2Dに設定

## 🎮 操作方法

### プレイヤーの操作
- **アクションスペースをクリック**: ワーカーを配置
- **カードをクリック**: カードをプレイ
- **ターン終了ボタン**: 手動でターン終了
- **ゲーム再開ボタン**: ゲームをリスタート

### ゲームルール（Agricola風）
- 各プレイヤーは家族2人から開始
- 家族を配置するとそのアクションが実行される
- 家族の成長により最大5人まで増加可能
- 収穫時に食料不足だと乞食カード（-3点）を受け取る
- 14ラウンド終了時の最高得点者が勝利
- スコアはバランス重視：家族・住居・畑・動物・作物など全分野で得点

## 🔧 カスタマイズ

### 新しいカードの追加
1. `Card.cs` を継承したクラスを作成
2. 特殊効果は `ExecuteSpecialEffect` メソッドをオーバーライド
3. ScriptableObject として作成し、デッキに追加

### 新しいアクションスペースの追加
1. シーンに新しいGameObjectを作成
2. `ActionSpace` コンポーネントを追加
3. アクションタイプとリソース効果を設定

### AIの強化
1. `GameManager.cs` の `ExecuteSimpleAI` メソッドを修正
2. より複雑な戦略ロジックを実装
